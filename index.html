shutt is built with integrity and technology in mind

Shutt 是觀念先進、靈活的 CSS Framework。提供團隊快使啟動專案，以及實現美學與創意的工具。同時簡化繁瑣工作流程，讓大型專案維護更簡潔。

跨瀏覽器

shutt 可在各個標準瀏覽器完美顯示，包含: Chrome, Firefox, Safari, Opera and IE9+.

授權

使用 shutt 需遵照 MIT 授權規定， shutt Framework 目前僅授權 [浩鑫股份有限公司-應用/智慧/雲端/使用者互動設計部] 使用.

Details

line height

基本的文字元素基礎行高line-height: 1.618em;，無論使用字體，套用這個值是我認為最完美的科學比例 。

box-sizing

所有的元素皆使用box-sizing: border-box;屬性，優點就是不需要再額外計算元素實際尺寸。

variables.sass

開發版使用variables.sass檔案. 只需透過變數設定即可快速誠心打造客製的全域基礎樣式.

grid

重新思考現有的 grid system 使用經驗中，必須是要簡單而通用的。

Basic template 

<!DOCTYPE html>
<html>
<head>
    <title>shutt Basic</title>

    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <link rel="stylesheet" href="css/shutt.min.css" />
    <link rel="stylesheet" href="css/your-styles.css" />
</head>
<body>
    <h1>Hello, world!</h1>
</body>
</html>
