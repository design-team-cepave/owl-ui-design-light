Mixins

clearfix

Clearfix is a solution to floating elements remain within the parent element.

Write

.element {
    .clearfix;
}

Get

.element:after {
    content: ".";
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
}

opacity

Sets the opacity. 0 is fully transparent, 100 is opaque.

Write

.element {
    .opacity(50);
}

Get

.element {
    filter: alpha(opacity = 50);
    -moz-opacity: 0.5;
    opacity: 0.5;
}

selection

Allows you to alter text selection on a page.

Write

.selection(255, 255, 158);

Get

::selection { background: rgb(255, 255, 158); }
::-moz-selection { background: rgb(255, 255, 158); }
img::selection { background: transparent; }
img::-moz-selection { background: transparent; }

box-sizing

Allows you to alter the CSS box model.

Write

.element {
    .box-sizing(border-box);
}

Get

.element {
    -webkit-box-sizing: border-box;
    -moz-box-sizing: border-box;
    box-sizing: border-box;
}

gradient

Gradient background. The first and second colors, going from bottom to top.

Write

.element {
    .gradient(#fff, #eee);
}

Get

.element {
    background-color: mix(#fff, #eee, 60%);
    background-image: -moz-linear-gradient(top, #fff, #eee); // FF 3.6+
    background-image: -ms-linear-gradient(top, #fff, #eee); // IE10
    background-image: -webkit-gradient(linear, 0 0, 0 100%, from(#fff), to(#eee)); // Safari 4+, Chrome 2+
    background-image: -webkit-linear-gradient(top, #fff, #eee); // Safari 5.1+, Chrome 10+
    background-image: -o-linear-gradient(top, #fff, #eee); // Opera 11.10
    background-image: linear-gradient(top, #fff, #eee); // The standard
    filter: e(%("progid:DXImageTransform.Microsoft.gradient(startColorstr='%d', endColorstr='%d', GradientType=0)",#fff,#eee)); // IE9 and down
}

bordered

Set a 1 pixel thick border that varies its color (top, right, bottom, left) on each side.

Write

.element {
    .bordered(#eee, #f2f2f2, #ddd, #e2e2e2);
}

Get

.element {
    border-top: 1px solid #eee;
    border-right: 1px solid #ddd;
    border-bottom: 1px solid #e2e2e2;
    border-left: 1px solid #f2f2f2;
}

retina-image

Set a retina image: url, width, height (optional).

Write

.element {
    .retina-image('logo.png', 100px);
}

Get

.element {
    background-image: url('/logo.png');
}
@media only screen and (-webkit-min-device-pixel-ratio: 2), only screen and (min--moz-device-pixel-ratio: 2), only screen and (-o-min-device-pixel-ratio: 2/1), only screen and (min-device-pixel-ratio: 2), only screen and (min-resolution: 192dpi), only screen and (min-resolution: 2dppx) {
    .element {
        background-image: url('/logo@2x.png');
        background-size: 100px auto;
    }
}

arrow-top

Sets the top arrow of element: color, size, left position (optional, default is 5px).

Write

.element {
    .arrow-top(#ddd, 10px);
}

Get

.element {
    position: relative;
}
.element:after {
    bottom: 100%;
    border: solid transparent;
    content: " ";
    height: 0;
    width: 0;
    position: absolute;
    pointer-events: none;
    border-bottom-color: #ddd;
    border-width: 10px;
    left: 5px;
}
